# 改卷和成绩查看功能使用说明

## 已创建的前端页面

### 1. 教师改卷页面
**文件**: `vue/src/views/manager/Grading.vue`
**路径**: `/manager/grading`
**功能**:
- 教师先选择课程（显示所有教授的课程及答卷数量）
- 进入课程后查看该课程下所有学生答卷列表
- 支持按状态筛选（待评分/AI已评分/已完成）
- 支持按学生姓名搜索
- 点击"批改"查看答卷详情
- 支持修改每道题的分数
- 支持一键接受AI评分结果
- 客观题（单选/多选/判断）自动评分并显示
- 主观题（填空/简答）显示AI评分，可人工修改

**页面流程**:
```
教师登录后台 → 点击"学生答卷"菜单 → 选择课程 → 查看答卷列表 → 点击批改 → 逐题评分/一键确认AI评分
```

### 2. 学生成绩页面
**文件**: `vue/src/views/front/MyScore.vue`
**路径**: `/front/myScore`
**功能**:
- 显示成绩统计卡片（考试次数、平均分、最高分、及格率）
- 显示最近3次考试
- 完整成绩列表，支持按状态筛选
- 点击"查看详情"可看到：
  - 每道题的得分情况
  - 学生自己的答案
  - 正确答案（选择题）或参考答案（主观题）
  - 答题正确/错误状态标记

**页面流程**:
```
学生登录前台 → 点击"我的成绩"菜单 → 查看统计和列表 → 点击查看详情 → 查看答题详情
```

## 后端API依赖

### 需要实现的后端接口

1. **获取教师课程列表**
   ```
   GET /score/courses?teacherId=1
   ```

2. **获取答卷列表（支持按课程筛选）**
   ```
   GET /score/list?teacherId=1&courseId=1
   ```

3. **获取答卷详情**
   ```
   GET /score/detail/{scoreId}
   ```

4. **人工改卷（单题）**
   ```
   POST /score/grade/{answerId}
   Body: { score: 8 }
   ```

5. **批量改卷**
   ```
   POST /score/batch/{scoreId}
   Body: { scores: [{answerId: 1, score: 8}, ...] }
   ```

6. **确认AI评分**
   ```
   POST /score/confirm-ai/{scoreId}
   ```

7. **获取学生成绩列表**
   ```
   GET /score/myScores?studentId=1
   ```

## 快速测试

### 测试教师改卷
1. 使用教师账号登录
2. 进入后台管理
3. 点击左侧菜单"学生答卷"
4. 选择一个课程
5. 点击某学生的"批改"按钮
6. 修改分数或点击"接受AI评分"

### 测试学生查看成绩
1. 使用学生账号登录
2. 点击顶部导航"我的成绩"
3. 查看统计卡片和成绩列表
4. 点击"查看详情"查看答题情况

## 注意事项

1. **角色权限**:
   - 教师改卷页面：仅TEACHER和ADMIN角色可访问
   - 学生成绩页面：仅STUDENT角色可访问

2. **API返回格式**:
   - 所有API应返回统一格式: `{ code: '200', data: ..., msg: '...' }`

3. **状态说明**:
   - `grading`: 待评分
   - `ai_graded`: AI已评分，等待人工确认
   - `finished`: 已完成

4. **题目类型**:
   - 1: 单选题（客观题，自动评分）
   - 2: 多选题（客观题，自动评分）
   - 3: 判断题（客观题，自动评分）
   - 4: 填空题（主观题，AI评分）
   - 5: 简答题（主观题，AI评分）
